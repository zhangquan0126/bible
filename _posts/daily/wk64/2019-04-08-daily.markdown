## 默想：神的故事
+ 大衛年紀老邁,日子滿足,就立他兒子所羅門做以色列的王。
+ 利未人從二十歲以外的都被數點,都辦耶和華殿的事務。
+ 亞倫子孫的班次。
+ 利未其他後裔亦掣籤得職任。

### 默想：我的故事
+ 【服事忠誠】大衛不僅預備了建築聖殿的材料,而且還預備了在聖殿中服事的人員。我們在教會的服事中應當如何兼顧物資和工人的需要?
+ 【服事忠誠】聖殿的職任可以分為哪些方面?今天我們的教會服事中有哪些類似的事奉?

### 禱告：

+ 感谢主的拣选，求主继续带领并祝福我。

### 筆記與回應：
